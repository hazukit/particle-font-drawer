Particle Font Drawer Demo

Simple Code Example


      new PirticleFontDrawer('top_canvas1', 'Click Here', {
        fontStyles: {'font-family':'Times New Roman', 'font-size': '100px'}
      });

Use Web Font


      new PirticleFontDrawer('top_canvas2', 'Web Font', {
        fontStyles: {'font-family':'Candal', 'font-size': '100px'},
        fontUrl: 'https://fonts.googleapis.com/css?family=Candal',
        textColor: '87ceeb',
        firstBound: false
      });

Set Particle Color From An Image


      new PirticleFontDrawer('top_canvas3', 'Iron Sand', {
        fontStyles: {'font-family':'Times New Roman', 'font-size': '100px'},
        textureUrl: 'img/bg.jpg'
      });
